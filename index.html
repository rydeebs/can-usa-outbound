Azure App Client ID
Pawel registers this himself — no IT admin needed. 5 minutes. 1. Sign in to portal.azure.com with CAN USA Microsoft account.
2. Azure Active Directory → App registrations → New registration.
3. Name: "CAN USA Outbound"  |  Redirect URI: Single-page application (SPA) → paste your Netlify URL.
4. Copy the Application (client) ID → paste above → Save.
5. API permissions → Add → Mail.Send + Mail.ReadWrite (Delegated).
6. Click "Sign in with Microsoft."

No Azure yet? Use Copy or Open in Outlook in the Review queue and Sequence Studio — both work without any setup.
